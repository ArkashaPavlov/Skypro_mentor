При выполнении работы найдены и исправлены следующие ошибки:

Файл index.html.
1. Текст внизу формы “© 202 4. SkyPro”, зачем-то пробел перед цифрой 4
2. Кнопки. Нажатие на них работает только если нажать именно на текст, а на остальную площадь не работает


Файл cmd.html.
1. Текст внизу формы “© 202 4. SkyPro”, зачем-то пробел перед цифрой 4
2. DIR – Неправильное описание команды - правильно будет "Отображение файлов и папок в текущем каталоге”
3. CD – Неправильное описание команды - правильно будет изменение текущего каталога.
4. MAKEDIR – Такой команды в принципе нету, чтобы создать папку надо писать mkdir
5. DATE – Неправильное описание команды - она используется для работы с текущей датой
6. TIME – Неправильное описание команды - она используется для работы со временем
7. Отсутствует кнопка возврата в меню

Файл mac.html.
1. ls - расшифровывается как "List", а не "List files in the directory". Правильный перевод: "перечислить" или "вывести на экран список файлов и папок в текущем каталоге".
2. mv - Неправильно. mv  расшифровывается как "Move". Перевод: "переместить" Так-же её можно использовать для переноса с переименованием файла
3. mkdir - расшифровывается как "Make directory", а не "Move". Перевод: "создать каталог" или "создать папку".
4. Кнопка возврата в меню так-же работает только если нажимать на текст а не на всё поле кнопки
5. В названии страницы пропущена буква "c"

Файл git.html
1. Команды дублируют ошибочное описание MacOS
2. Кнопка Возврата в меню отсылает на информацию о MacOS, а не в главное меню, и так-же работает только на тексте
3. Текст внизу формы “© 202 4. SkyPro”, зачем-то пробел перед цифрой 4
